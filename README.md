# canvas-drawing-board
学习中使用canvas做的一个小画板

## 移动端预览
<img src="https://github.com/lynchuh/canvas-drawing-board/blob/master/preview.png" style="height:60px;width:60px;position:absolute;right:70px;margin-top:-10px">

## 实现功能

- 适配PC/移动端
- 选择画笔/橡皮擦
- 选择画笔粗细/颜色
- 一键删除所有画面
- 保存涂鸦到本地



## 使用到的技术点
- 使用jQuery实现事件委托
- 使用this/闭包 实现数据封装与隔离
- 使用canvas实现绘图功能
